# App Spec: Notes (/notes)

## Концепция
Минималистичный редактор заметок с поддержкой Markdown и WYSIWYG-интерфейсом ("Notion-style").

## Функциональные требования
1.  **Интерфейс:**
    *   Двухпанельный макет: слева список всех заметок, справа поле редактора.
    *   Редактор: WYSIWYG по умолчанию (ввод `# ` превращается в заголовок и т.д.).
    *   Режим "Raw": переключатель для просмотра/редактирования чистого Markdown текста.
2.  **Управление заметками:**
    *   Автосохранение при вводе (debounce) и при переключении между заметками.
    *   Бесконечный список заметок (боковая панель).
    *   Создание новой заметки, удаление существующих.
3.  **Авторизация и доступ:**
    *   Вход через Google (интеграция с Cloudflare Access/Zero Trust).
    *   Заметки привязаны к `user_id`.
    *   Доступ только для владельца и круга "FFF" (Friends, Family, Founders).
4.  **Хранение данных:**
    *   База данных для хранения текстов заметок и метаданных.
    *   API для синхронизации фронтенда с бэкендом.

## Технологический стек (предложение)
- **Frontend:** React + Tailwind CSS.
- **Editor Engine:** TipTap или Lexical (лучшие для Notion-style опыта).
- **Backend:** Node.js (Fastify/Express).
- **DB:** PostgreSQL (в Docker).
