# WoWs Planner (Планшет)

Инструмент для тактического планирования боев в «Мире кораблей». Поддерживает совместную работу в реальном времени, рисование тактических схем и визуализацию зон действия снаряжения.

## Возможности

- **Совместная работа:** Несколько пользователей могут одновременно редактировать один планшет. Видны действия всех участников (перемещение кораблей, рисование, пинги).
- **Тактические слои:** Поддержка нескольких слоев (сценариев) внутри одного планшета. Каждому слою можно задать масштаб в километрах.
- **Инструменты рисования:**
  - **Стрелки наступления:** Плавные сужающиеся стрелки с градиентом для обозначения маневров.
  - **Тактические круги:** Произвольные круги с указанием радиуса в километрах.
  - **Перо:** Свободное рисование.
- **Корабли:** Расстановка иконок кораблей разных классов (Линкоры, Крейсеры, Эсминцы, Авианосцы, Подлодки).
- **Визуализация зон:** Для каждого корабля можно включить отображение радиусов РЛС (синий) и ГАП (зеленый).
- **Пинги:** Быстрая сигнализация на карте кликом мыши (исчезающие круги выбранного цвета).
- **Фоновые карты:** Возможность загрузки изображений карт на каждый слой.

## Технологии

- **Frontend:** React 19, Vite, Tailwind CSS 4, Lucide Icons, TanStack Table (в Shiptools).
- **Backend:** Node.js, Fastify, PostgreSQL (с использованием JSONB для хранения состояния).
- **Real-time:** WebSockets для синхронизации действий пользователей.

## Архитектура данных

Состояние планшета хранится в базе данных PostgreSQL в виде JSON-объекта. При любом изменении (перемещение, рисование) фронтенд отправляет обновленное состояние на бэкенд, который рассылает его всем подключенным пользователям через сокеты.

## Разработка

### Локальный запуск (Development)

1. **База данных:**
   ```bash
   docker compose -f docker-compose.local-test.yml up -d db
   ```

2. **Бэкенд:**
   ```bash
   cd wows_planner/backend
   npm install
   DATABASE_URL=postgres://user:password@localhost:5432/wows_planner npm run dev
   ```

3. **Фронтенд:**
   ```bash
   cd wows_planner/frontend
   npm install
   npm run dev
   ```
   Откройте `http://localhost:5173/wows_planner/`

### Деплой

Деплой автоматизирован через GitHub Actions. При создании тега `v*` происходит сборка Docker-образов и обновление кода на сервере.
